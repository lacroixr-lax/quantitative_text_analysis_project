# quantitative_text_analysis_project
Project Overview:
This project applies quantitative text analysis techniques to a dataset of investigative legal documents. The goal was to explore language patterns and topic structures using computational text analysis methods.

Methods used: 
- Tokenisation
- Term frequency analysis
- Topic modelling
- Data visualisation

Tools:
- R
- tidyverse
- stringr
- topicmodels
- ggraph
- spacyr
